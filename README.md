# DigiTools

## Project Name
DigiTools - A Digital Tools Platform

## Description
This is a simple digital tools website where users can explore different premium tools and add them to a cart. Users can easily switch between the product section and the cart section. The main goal of this project is to provide a clean UI and smooth user experience.


## Features
- Add products to cart and remove them  
- Toggle between Products and Cart section  
- Button changes and shows alert when product is added
